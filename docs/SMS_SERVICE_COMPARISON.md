# SMS 알림 서비스 비교 및 선택 가이드

## 📱 개요

상담문의 및 특강신청 시 관리자에게 즉시 SMS 알림을 보내기 위한 서비스 선택 가이드입니다.

## 🎯 주요 SMS 서비스 비교

### 1. 알리고 (Aligo) ⭐ 추천

**장점:**
- ✅ 한국 서비스 (국내 최대 시장 점유율)
- ✅ 저렴한 가격 (건당 약 20원)
- ✅ 안정적인 서비스
- ✅ 간단한 API 구조
- ✅ 080 번호 무료 발신 가능
- ✅ 즉시 가입 가능

**단점:**
- ❌ 해외 발신 불가

**가격:**
- 일반 SMS: 약 20원/건
- LMS (장문): 약 50원/건
- 무료 충전 제공 (신규 가입 시)

**API 문서:**
- [알리고 개발자 센터](https://smartsms.aligo.in/)

**적합한 경우:**
- 국내 서비스 운영
- 비용 효율성 중요
- 빠른 구현 필요

---

### 2. 카카오톡 알림톡

**장점:**
- ✅ 브랜드 이미지 향상 (카카오톡 메시지)
- ✅ 이미지/버튼 포함 가능
- ✅ 개인정보보호 (수신자 번호 마스킹)
- ✅ 무료 템플릿 제공

**단점:**
- ❌ 비싼 가격 (건당 약 50원)
- ❌ 템플릿 사전 승인 필요
- ❌ 카카오 비즈니스 계정 필요
- ❌ 설정 복잡

**가격:**
- 알림톡: 약 50원/건
- 친구톡: 약 100원/건

**적합한 경우:**
- 브랜드 이미지 중요
- 템플릿 메시지 사용
- 마케팅 목적

---

### 3. Twilio

**장점:**
- ✅ 글로벌 서비스
- ✅ 다양한 국가 지원
- ✅ 안정적인 API
- ✅ 실시간 전송 상태 확인

**단점:**
- ❌ 한국 가격 비쌈 (약 200원/건)
- ❌ 해외 서비스 (지원 한국어 제한)
- ❌ 결제 복잡 (신용카드만)

**가격:**
- 한국 SMS: 약 200원/건

**적합한 경우:**
- 해외 발신 필요
- 글로벌 서비스 운영

---

### 4. NHN Cloud (Toast)

**장점:**
- ✅ 한국 기업 서비스
- ✅ 안정적인 인프라
- ✅ 다양한 연동 옵션

**단점:**
- ❌ 비싼 가격 (건당 약 30원)
- ❌ 기업용에 최적화
- ❌ 설정 복잡

**가격:**
- SMS: 약 30원/건

---

## 💡 추천: 알리고 (Aligo)

**이유:**
1. **비용 효율성**: 가장 저렴한 가격 (건당 20원)
2. **구현 용이성**: 간단한 API, 빠른 연동
3. **국내 서비스**: 한국어 지원, 안정적
4. **신규 가입**: 즉시 시작 가능, 무료 충전 제공

**예상 월 비용:**
- 상담문의 100건/월: 약 2,000원
- 특강신청 50건/월: 약 1,000원
- **총 예상 비용: 약 3,000원/월**

## 📋 다음 단계

1. **알리고 계정 생성** (예상 시간: 30분)
   - [알리고 회원가입](https://smartsms.aligo.in/)
   - 본인인증 완료
   - 무료 충전 확인

2. **API 키 발급** (예상 시간: 10분)
   - 알리고 관리자 페이지
   - API 인증키 발급
   - 발신번호 등록

3. **코드 연동** (예상 시간: 2-3시간)
   - `app/api/consultations/route.ts` 수정
   - `app/api/airplane-registration/route.ts` 수정
   - 에러 처리 추가

4. **환경 변수 설정** (예상 시간: 10분)
   - Vercel 환경 변수 추가
   - 로컬 `.env.local` 설정

5. **테스트** (예상 시간: 1시간)
   - 실제 SMS 발송 테스트
   - 다양한 시나리오 테스트

## 🔄 다른 서비스 선택 시

위의 추천을 따르지 않고 다른 서비스를 선택하는 경우:

1. 해당 서비스의 API 문서 확인
2. `lib/sms.ts` 파일 수정 필요
3. 환경 변수 키 이름 변경 가능
4. 기본 구조는 동일하게 유지
