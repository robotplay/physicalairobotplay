## 📋 다음 단계: 관리자 페이지 개선

### 1단계: 관리자 페이지에 결제 내역 조회 기능 추가 (우선)

**현재 상태:**
- 관리자 페이지는 상담 문의만 표시 (LocalStorage 기반)
- 결제 내역은 MongoDB에 저장되지만 관리자 페이지에서 볼 수 없음

**필요한 작업:**
- [ ] 결제 내역 조회 API 생성 (`/api/payments`)
- [ ] 관리자 페이지에 결제 내역 탭 추가
- [ ] 결제 내역 목록 표시
- [ ] 결제 상세 정보 표시
- [ ] 신청서와 결제 내역 연결 표시

**예상 소요 시간:** 1-2시간

### 2단계: 관리자 페이지를 MongoDB 기반으로 전환

**현재 상태:**

- 신청서는 MongoDB에 저장되지만 관리자 페이지에서 조회 안 됨

**필요한 작업:**
- [ ] 상담 문의 조회 API 생성 (`/api/consultations`)
- [ ] 신청서 조회 API 생성 (`/api/airplane-registrations`)
- [ ] 관리자 페이지를 MongoDB에서 데이터 가져오도록 변경
- [ ] 실시간 업데이트 기능 추가

**예상 소요 시간:** 1-2시간

### 3단계: 통합 대시보드 생성
