# MongoDB Atlas 프로젝트 삭제 가이드

## ⚠️ 주의사항

**프로젝트를 삭제하면:**
- 프로젝트 내의 모든 클러스터가 삭제됩니다
- 모든 데이터가 영구적으로 삭제됩니다
- 복구할 수 없습니다

**삭제 전 확인:**
- [ ] 프로젝트 내의 중요한 데이터가 없는지 확인
- [ ] 다른 프로젝트로 데이터를 백업했는지 확인
- [ ] 정말 삭제가 필요한지 확인

---

## 프로젝트 삭제 방법

### 방법 1: Actions 컬럼의 휴지통 아이콘 사용

1. **MongoDB Atlas 대시보드 접속**
   - https://cloud.mongodb.com 접속
   - 로그인

2. **프로젝트 목록 확인**
   - 왼쪽 상단의 프로젝트 이름 클릭
   - "All Projects" 또는 프로젝트 목록 확인

3. **"Academy" 프로젝트 찾기**
   - 프로젝트 목록에서 "Academy" 찾기

4. **삭제 아이콘 클릭**
   - "Academy" 프로젝트 행의 오른쪽 "Actions" 컬럼
   - **휴지통 아이콘** 클릭

5. **삭제 확인**
   - 확인 대화상자가 나타남
   - 프로젝트 이름을 입력하여 확인 (예: `Academy`)
   - "Delete" 또는 "삭제" 버튼 클릭

---

### 방법 2: 프로젝트 설정에서 삭제

1. **"Academy" 프로젝트 선택**
   - 프로젝트 목록에서 "Academy" 클릭

2. **Settings 메뉴로 이동**
   - 왼쪽 메뉴에서 "Settings" 클릭
   - 또는 프로젝트 이름 옆 "..." 메뉴 → "Settings"

3. **프로젝트 삭제**
   - Settings 페이지 하단으로 스크롤
   - "Delete Project" 또는 "프로젝트 삭제" 섹션 찾기
   - "Delete Project" 버튼 클릭

4. **삭제 확인**
   - 확인 대화상자에서 프로젝트 이름 입력
   - "Delete" 버튼 클릭

---

## 삭제 과정

### Step 1: 삭제 시작

1. **휴지통 아이콘 또는 "Delete Project" 클릭**
2. **확인 대화상자 표시**

### Step 2: 확인 입력

**일반적으로 다음 중 하나를 요청합니다:**
- 프로젝트 이름 입력 (예: `Academy`)
- "DELETE" 또는 "삭제" 입력
- 체크박스 확인

### Step 3: 삭제 완료

1. **"Delete" 또는 "삭제" 버튼 클릭**
2. **삭제 진행 중 표시**
3. **삭제 완료 메시지 표시**
4. **프로젝트 목록에서 제거됨**

---

## 삭제 후 확인

### 삭제 완료 확인:

1. **프로젝트 목록 확인**
   - "Academy" 프로젝트가 목록에서 사라졌는지 확인

2. **다른 프로젝트 선택**
   - 다른 프로젝트로 전환하여 정상 작동 확인

---

## 주의사항

### 삭제 전 백업 (권장)

**중요한 데이터가 있다면:**

1. **데이터 내보내기**
   - Database → Collections
   - 데이터 내보내기 (Export)

2. **다른 프로젝트로 마이그레이션**
   - 새 프로젝트 생성
   - 데이터 가져오기 (Import)

### 클러스터가 있는 경우

**프로젝트에 클러스터가 있으면:**
- 클러스터도 함께 삭제됩니다
- 클러스터 삭제는 되돌릴 수 없습니다

---

## 문제 해결

### 삭제 버튼이 비활성화된 경우

**가능한 원인:**
- 프로젝트에 활성 클러스터가 있음
- 프로젝트에 활성 서비스가 있음
- 권한이 부족함

**해결 방법:**
1. **모든 클러스터 삭제**
   - Database → 클러스터 삭제
2. **모든 서비스 종료**
   - 프로젝트 내 모든 서비스 확인
3. **권한 확인**
   - Organization Owner 권한 확인

### 삭제가 진행되지 않는 경우

1. **브라우저 새로고침**
2. **다른 브라우저에서 시도**
3. **MongoDB Atlas 지원팀 문의**

2. ✅ Actions 컬럼의 휴지통 아이콘 클릭
3. ✅ 확인 대화상자에서 프로젝트 이름 입력
4. ✅ "Delete" 버튼 클릭
5. ✅ 삭제 완료 확인

**⚠️ 중요:**
- 삭제 전 데이터 백업 권장
- 삭제는 되돌릴 수 없음
- 모든 클러스터와 데이터가 삭제됨
