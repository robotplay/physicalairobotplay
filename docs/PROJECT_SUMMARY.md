## 🏗️ 시스템 아키텍처

### 전체 플로우

```
사용자 수강 신청
    ↓
이메일 + 접근 코드 생성
    ↓
이메일 발송 (링크 + 코드)
    ↓
사용자 선택:
  - 이메일 링크 클릭 → 즉시 수강
  - 코드 입력 → 검증 → 수강
    ↓
JWT 토큰 검증
    ↓
강좌 콘텐츠 접근
```

### 기술 스택

**Frontend:**
- Next.js 16 (App Router)
- TypeScript
- Tailwind CSS
- React Hot Toast
- Tiptap (Rich Text Editor)

**Backend:**
- Next.js API Routes
- MongoDB 7
- JWT (jose)
- bcryptjs

**Infrastructure:**
- Vercel (Hosting)
- MongoDB Atlas (Database)
- Gmail SMTP (Email)
- PortOne (Payment)
