# 사이트 고도화 가이드

## 개요

parplay.co.kr 사이트의 학부모 타겟팅 강화, 전문성 강화, 운영 시스템 고도화를 위한 개선사항이 적용되었습니다.

## 구현된 기능

### 1. 학부모 타겟팅 강화

#### ✅ "Why PAR Play?" 섹션 (Mom's View)
- **위치**: 메인 페이지 (`/#moms-view`)
- **기능**: 기술 용어를 교육 효과로 번역하여 학부모가 이해하기 쉽게 설명
- **컴포넌트**: `components/MomsView.tsx`
- **주요 내용**:
  - Python/C++ → 논리적 사고력과 문제 해결 능력
  - 자율주행/비전인식 → 미래 AI 시대를 리드할 핵심 소양
  - 대회 수상 → 아이의 자존감 향상 및 입시 포트폴리오 완성
  - 학부모 후기 3건 포함

#### ✅ 후기 및 결과물 갤러리 강화
- **위치**: 메인 페이지 (`/#success`)
- **기능**: 학생 후기와 함께 학부모 후기 추가 표시
- **컴포넌트**: `components/SuccessStories.tsx` (수정됨)
- **개선사항**:
  - 각 학생 후기 카드에 학부모님의 짧은 한 줄 후기 추가
  - 수상 사진과 함께 감성적인 스토리텔링

#### ✅ 안전/안심 강조 FAQ
- **위치**: 메인 페이지 (`/#safety`)
- **기능**: 학부모가 걱정하는 안전 문제에 대한 명확한 답변
- **컴포넌트**: `components/SafetyFAQ.tsx`
- **주요 FAQ**:
  - 드론 수업의 안전성
  - 사용 장비 및 소재의 안전성
  - 수업 환경의 쾌적성
  - 보험 가입 여부
  - 감염병 예방 조치
  - 초등학생 안전 수업 가능 여부

### 2. 전문성 및 권위 강화

#### ✅ 대표 프로필 디테일 업
- **위치**: 메인 페이지 (`/#leader`)
- **기능**: 하광진 대표님의 이력, 자격증, 교육 철학 상세 소개
- **컴포넌트**: `components/LeaderProfile.tsx`
- **주요 내용**:
  - 타임라인 형태의 주요 이력 (2020년 대통령상 수상부터 현재까지)
  - 교육 철학 및 핵심 가치
  - 주요 경력 및 자격증
  - 지도 학생 수, 대회 수상 통계

#### ✅ 커리큘럼 로드맵 시각화 강화
- **위치**: 메인 페이지 (`/#roadmap`)
- **기능**: 입문 → 기초 → 심화 → 대회반 → 영재/진학 단계별 로드맵
- **컴포넌트**: `components/EnhancedRoadmap.tsx`
- **주요 내용**:
  - 5단계 성장 경로 시각화
  - 각 단계별 기대 성과 명시
  - 기간 및 설명 포함
  - 장기 비전 제시 (1년 후, 3년 후)

### 3. 사이트 운영 시스템 고도화

#### ✅ 실시간 상담 채널 연동
- **위치**: 모든 페이지 우측 하단 플로팅 버튼
- **기능**: 카카오톡, 네이버 톡톡, 전화 상담 원클릭 접근
- **컴포넌트**: `components/ChatButton.tsx`
- **설정 방법**:
  1. `.env.local` 파일에 다음 환경 변수 추가:
     ```
     NEXT_PUBLIC_KAKAO_CHANNEL_URL=https://pf.kakao.com/_your_channel_id
     NEXT_PUBLIC_NAVER_TALK_URL=https://talk.naver.com/your_channel_id
     ```
  2. 카카오톡 채널 관리자 페이지에서 채널 URL 복사
  3. 네이버 톡톡 관리자 페이지에서 채널 URL 복사
  4. 환경 변수에 실제 URL 입력

#### ✅ 공지사항 및 블로그 기능
- **위치**: `/news` 페이지
- **기능**: 최신 소식, 공지사항, 교육 정보 게시
- **파일**: `app/news/page.tsx`
- **현재 상태**: 정적 데이터로 구현 (추후 CMS 연동 가능)
- **향후 개선**:
  - MongoDB에 공지사항 저장
  - 관리자 페이지에서 공지사항 작성/수정/삭제 기능
  - 카테고리별 필터링
  - 검색 기능

## 네비게이션 업데이트

헤더 메뉴에 다음 항목이 추가되었습니다:
- 대표 소개 (`/#leader`)
- 학부모 전용 (`/#moms-view`)
- 커리큘럼 (`/#roadmap`)
- 안전 FAQ (`/#safety`)
- 소식 (`/news`)

## 환경 변수 설정

`.env.local` 파일에 다음 변수를 추가하세요:

```env
# 상담 채널 URL
NEXT_PUBLIC_KAKAO_CHANNEL_URL=https://pf.kakao.com/_your_channel_id
NEXT_PUBLIC_NAVER_TALK_URL=https://talk.naver.com/your_channel_id
```

## 다음 단계 제안

### 단기 (1-2주)
1. ✅ 카카오톡 채널 및 네이버 톡톡 채널 생성 및 URL 설정
2. ✅ 실제 학부모 후기 수집 및 업데이트
3. ✅ 대표님 프로필 사진 및 상세 이력 업데이트
4. ✅ 공지사항 첫 게시물 작성

### 중기 (1-2개월)
1. 공지사항 CMS 연동 (MongoDB + 관리자 페이지)
2. 블로그 SEO 최적화
3. 소셜 미디어 연동 (페이스북, 인스타그램 피드)
4. 이메일 뉴스레터 구독 기능

### 장기 (3-6개월)
1. 온라인 강의 플랫폼 연동
2. 학생 포트폴리오 갤러리
3. 실시간 수업 일정 캘린더
4. 학부모 전용 대시보드 (자녀 진도 확인)

## 테스트 체크리스트

- [ ] 메인 페이지 모든 섹션이 정상 표시되는지 확인
- [ ] 학부모 전용 섹션의 기술→혜택 번역이 적절한지 확인
- [ ] 안전 FAQ가 학부모의 주요 걱정을 다루는지 확인
- [ ] 대표 프로필의 이력이 정확한지 확인
- [ ] 커리큘럼 로드맵이 명확하게 전달되는지 확인
- [ ] 실시간 상담 버튼이 모든 페이지에서 작동하는지 확인
- [ ] 카카오톡/네이버 톡톡 링크가 올바른지 확인
- [ ] 공지사항 페이지가 정상 작동하는지 확인
- [ ] 모바일 반응형이 정상인지 확인
- [ ] 페이지 로딩 속도 확인
